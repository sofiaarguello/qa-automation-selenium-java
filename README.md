# QA Automation Project - Selenium (Java)

This project contains automated tests for web applications using Selenium WebDriver and Java.

## 🚀 About the project
I created this project to practice automation testing and improve my QA skills, focusing on real user flows and UI validation.

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- GitHub Actions (basic CI)

## 🧩 Architecture
The project follows the Page Object Model (POM) to organize code and separate test logic from page elements.

## ✅ Test Scenarios
- Login and logout functionality
- Form validations
- Navigation flows
- UI element verification

## ▶️ How to run
1. Clone the repository  
2. Open in IntelliJ or Eclipse  
3. Run the test classes  

## 🎯 What I learned
- Writing automated test cases  
- Using Selenium for UI testing  
- Structuring a project with POM  
- Running tests with TestNG
